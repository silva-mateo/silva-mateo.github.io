# Portfolio de Mateo Silva

Portfolio personal desarrollado con React, TypeScript y Vite.

## Características

- Diseño responsive.
- Soporte para tema claro y oscuro.
- Cambio de idioma entre español e inglés.
- Secciones de perfil, tecnologías, proyectos y contacto.
- Formulario de contacto integrado.
- Footer simple con enlaces sociales.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- pnpm

## Uso local

```bash
nvm use
corepack enable
pnpm install
pnpm dev
```

## Scripts

```bash
pnpm dev      # Entorno de desarrollo
pnpm build    # Build de producción
pnpm lint     # Lint del proyecto
pnpm preview  # Preview de la build
```
